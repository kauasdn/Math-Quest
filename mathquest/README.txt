Math Quest - Godot 4 Project
===========================

Como usar:
1. Baixe e extraia o arquivo .zip.
2. Abra o Godot 4.
3. No diálogo 'Import or create a project', selecione a pasta 'MathQuest' que contém project.godot.
4. O GameManager e QuestionDatabase já estão configurados como AutoLoads no project.godot.
5. Abra a cena 'scenes/MainMenu.tscn' e execute.

Observações:
- O projeto é básico e pronto para importar. Você pode trocar imagens e sons na pasta 'assets'.
- As perguntas já estão preenchidas (10 por reino).
- Se o Godot reclamar de paths ou autoload, vá em Project -> Project Settings -> Autoload e confirme que os scripts estão adicionados (GameManager e QuestionDatabase).

Boa diversão!
